HTML

  • The head element contains information about the webpage
  • The body represents the visible content shown to the user
  • Headers must be placed in numerical order
  • Html is the framework of a webpage providing structure for content

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much padding we want around the content inside of an element.
  • Internal style sheets contain CSS rules for the webpage inside of the head section of HTML
  • External style sheets keep CSS rules in seperate file that is linked to HTML file.
